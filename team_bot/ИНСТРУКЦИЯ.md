# 🤖 Team Manager Bot — Инструкция по установке

## Что умеет бот

| Функция | Описание |
|---|---|
| ☐ Посещаемость | В 10:30 вы получаете список команды с галочками. Снимаете отсутствующих → бот отправляет предупреждение в тему «Время работы» |
| ☀️ Git pull | В 10:00 бот напоминает в General стянуть изменения |
| 🌙 Git push + отчёт | В 20:00 бот напоминает запушить изменения и написать отчёт |
| 🤖 Анализ отчётов | В 21:00 Claude читает все отчёты и пишет: кто написал, кто нет, краткая выжимка |
| 📊 Аналитика | /analytics — статистика за 30 дней по каждому участнику |
| 🔔 Кастомные уведомления | Добавляйте свои уведомления прямо в боте |

---

## Шаг 1 — Создайте бота в Telegram

1. Откройте Telegram, найдите **@BotFather**
2. Напишите `/newbot`
3. Придумайте название (например: `Team Manager`)
4. Придумайте username (например: `myteam_manager_bot`)
5. Скопируйте **токен** — он выглядит так: `7123456789:AAHdq...`

---

## Шаг 2 — Получите Anthropic API Key

1. Зайдите на https://console.anthropic.com/
2. Зарегистрируйтесь (бесплатно)
3. Перейдите в **API Keys** → **Create Key**
4. Скопируйте ключ (начинается с `sk-ant-...`)

> 💡 Бесплатного кредита хватает на несколько месяцев при ежедневном анализе отчётов

---

## Шаг 3 — Узнайте ID группы и тем

### ID группы:
1. Добавьте **@userinfobot** в вашу группу
2. Напишите в группе `/start`
3. Бот покажет ID группы (например: `-1001234567890`)
4. После — удалите бота из группы

### ID тем (topics):
1. Откройте нужную тему в группе
2. Нажмите на тему — скопируйте ссылку
3. В ссылке `https://t.me/c/1234567890/42` → ID темы = **42**

### Ваш личный Telegram ID:
1. Напишите боту **@userinfobot** в личку `/start`
2. Он покажет ваш ID (число, например `987654321`)

---

## Шаг 4 — Заполните config.py

Откройте файл `config.py` и замените все значения:

```python
BOT_TOKEN = "7123456789:AAHdq..."      # ← ваш токен
ANTHROPIC_API_KEY = "sk-ant-..."       # ← ваш ключ
GROUP_ID = -1001234567890              # ← ID группы
TOPIC_WORK_TIME = 5                    # ← ID темы "Время работы"
TOPIC_GENERAL = 1                      # ← ID темы "General"
TOPIC_REPORTS = 8                      # ← ID темы "Отчёты"
ADMIN_IDS = [987654321]               # ← ваш Telegram ID
TIMEZONE = "Asia/Bishkek"             # ← ваш часовой пояс
```

---

## Шаг 5 — Добавьте бота в группу

1. Откройте вашу Telegram группу
2. Нажмите на название группы → **Добавить участника**
3. Найдите вашего бота по username
4. Добавьте его и **назначьте администратором** (нужно для отправки сообщений в темы)

---

## Шаг 6 — Запуск на компьютере (локально)

### Установите Python (если нет):
- Скачайте с https://python.org/downloads/
- При установке поставьте галочку **"Add to PATH"**

### Установите зависимости:
```bash
# Откройте терминал/командную строку в папке с ботом
pip install -r requirements.txt
```

### Запустите:
```bash
python bot.py
```

> ⚠️ При локальном запуске бот работает только пока открыт терминал.
> Для постоянной работы — см. Шаг 7.

---

## Шаг 7 — Постоянный запуск (бесплатно)

### Вариант A: Railway.app (рекомендую, бесплатно)

1. Зайдите на https://railway.app — зарегистрируйтесь через GitHub
2. Создайте **New Project → Deploy from GitHub**
3. Загрузите файлы бота на GitHub (или используйте локальную загрузку)
4. В настройках проекта добавьте переменные окружения:
   - `BOT_TOKEN` = ваш токен
   - `ANTHROPIC_API_KEY` = ваш ключ
   - И остальные из config.py
5. Railway сам запустит бота 24/7

### Вариант B: VPS (если хотите контроль)
Хороший дешёвый вариант — **Timeweb VPS** (~200 руб/мес) или **DigitalOcean** ($4/мес).

```bash
# На сервере:
git clone <ваш репозиторий>
cd team_bot
pip install -r requirements.txt

# Запуск как сервис (чтобы не останавливался):
nohup python bot.py &
```

## Шаг 8 — Добавьте команду участников

После запуска напишите боту в личку:

```
/addmember 123456789 Анна
/addmember 987654321 Борис
/addmember 555666777 Виктор
```

ID каждого участника они могут узнать сами через @userinfobot.

---

## Команды бота

| Команда | Что делает |
|---|---|
| `/attendance` | Открыть список посещаемости прямо сейчас |
| `/members` | Показать список команды |
| `/addmember <id> <Имя>` | Добавить участника |
| `/removemember` | Удалить участника (с кнопками) |
| `/notifications` | Управление уведомлениями (вкл/выкл) |
| `/addnotif <HH:MM> <топик> <текст>` | Добавить своё уведомление |
| `/analytics` | Статистика за 30 дней |
| `/report_now` | Запустить анализ отчётов прямо сейчас |

## Расписание уведомлений

| Время | Куда | Что |
|---|---|---|
| 10:00 | General | Git pull напоминание |
| 10:30 | → вам в личку | Список посещаемости с галочками |
| 20:00 | General | Напоминание об отчёте и git push |
| 21:00 | Отчёты | AI-анализ отчётов за день |

## Проблемы?

**Бот не отправляет сообщения в группу:**
- Убедитесь что бот — администратор группы
- Проверьте правильность GROUP_ID (должен быть отрицательным)

**Бот не пишет в темы:**
- В настройках группы должны быть включены Topics/Темы
- Проверьте правильность ID тем

**Ошибка при запуске:**
- Проверьте что все поля в config.py заполнены
- Убедитесь что установлены все зависимости: `pip install -r requirements.txt`
